01 O que é o DOM?

Quando o navegador carrega um arquivo HTML, ele constrói uma árvore de objetos na memória. Essa árvore é o Document Object Model (DOM). Cada tag HTML vira um nó que pode ser lido, alterado, criado ou removido via JavaScript — sem precisar recarregar a página.

árvore DOM simplificada 
document
 └── <html>
      ├── <head>
      │    └── <title>
      └── <body>
           ├── <h1>
           ├── <ul>
           │    ├── <li>
           │    └── <li>
           └── <button>

O objeto document é a porta de entrada: todo acesso ao DOM começa por ele.

💡 conceito-chave Manipular o DOM = acessar nós dessa árvore e mudar suas propriedades. A página atualiza na hora, sem reload.

02 Encontrar elementos

Antes de qualquer manipulação, precisamos selecionar o elemento. Existem cinco formas principais:

Por ID
getElementById('id')

Retorna 1 elemento exato. O mais rápido — use quando o elemento tem id único.

Por Tag
getElementsByTagName('li')

Retorna HTMLCollection (coleção "viva") de todos os elementos com aquela tag.

Por Classe
getElementsByClassName('cls')

Retorna HTMLCollection de todos os elementos com aquela classe CSS.

Moderno — 1 resultado
querySelector('.cls')

Aceita qualquer seletor CSS. Retorna o primeiro elemento que casar.

Moderno — vários
querySelectorAll('li')

Aceita qualquer seletor CSS. Retorna NodeList (tem .forEach()!).

⚠️ diferença importante HTMLCollection (getElementsBy…) não tem .forEach() diretamente. Use Array.from(colecao).forEach(…) ou prefira querySelectorAll, que retorna NodeList com suporte a .forEach().
exemplos de seleção 
// Pelo id — retorna 1 elemento
const campo = document.getElementById('novaTarefa');

// querySelector — seletor CSS, retorna o primeiro
const lista = document.querySelector('#listaTarefas');

// querySelectorAll — todos os <li> dentro da lista
const itens = document.querySelectorAll('#listaTarefas li');
itens.forEach(item => console.log(item.textContent));

✅ dica da prova Prefira textContent para texto simples — é mais seguro que innerHTML pois não executa HTML injetado. Use innerHTML apenas quando precisar inserir tags de fato.

textContent vs innerText — qual usar?

Ambas as propriedades leem e escrevem o texto de um elemento, mas têm comportamentos diferentes que podem pegar na prova:

CaracterísticatextContentinnerText
O que retorna Todo o texto do elemento, incluindo o de elementos ocultos (display:none) Só o texto visível na tela — respeita CSS
Tags filhas Ignora tags, retorna texto puro de tudo Interpreta a renderização: <br> vira \n, elementos ocultos são ignorados
Performance ✅ Mais rápido — não força reflow do layout ⚠️ Mais lento — o navegador precisa calcular o layout antes de responder
Segurança ✅ Nunca interpreta HTML — seguro contra XSS ✅ Também não interpreta HTML
Uso ideal Escrever texto em elementos (é o padrão na prova) Quando você precisa do texto exatamente como o usuário o vê
textContent vs innerText — exemplo prático 
// HTML: <div id="ex">Olá <span style="display:none">oculto</span> mundo</div>
const div = document.getElementById('ex');

console.log(div.textContent);  // → "Olá oculto mundo"  (inclui o span oculto)
console.log(div.innerText);     // → "Olá mundo"         (ignora o span oculto)

// Na ESCRITA, o comportamento é idêntico — define texto puro:
div.textContent = 'Novo texto';  // ✅ padrão recomendado na prova
div.innerText   = 'Novo texto';  // também funciona, mas evite por consistência
⚠️ resumo rápido para a prova Use sempre textContent para definir texto em elementos criados com JS. Reserve innerText para quando precisar ler o texto visível de um elemento que pode ter filhos ocultos. Na prática da prova (criar spans, botões, li), textContent é a escolha certa.

04 Criar e remover elementos

Fluxo de criação — 3 passos obrigatórios

1
Criar em memória com document.createElement('tag'). O elemento existe, mas ainda não está na página.
2
Configurar: definir texto, classes, atributos e event listeners.
3
Inserir na página com pai.appendChild(novo). Sem esse passo, o elemento é invisível — é o erro mais comum!
❌ erro clássico Esquecer o appendChild(). O elemento existe em memória mas nunca aparece na tela.
criar um <li> com texto e botão excluir 
function criarItemLista(texto, index) {
  // Passo 1 — criar
  const li  = document.createElement('li');
  const btn = document.createElement('button');

  // Passo 2 — configurar
  li.textContent = texto;
  btn.textContent = 'Excluir';
  btn.addEventListener('click', () => excluir(index));

  // Passo 3 — inserir
  li.appendChild(btn);                          // btn entra no li
  document.getElementById('lista').appendChild(li); // li entra na ul
}

Remover elementos

formas de remover 
// Forma moderna (recomendada) — remove a si mesmo
elemento.remove();

// Forma clássica — precisa do pai
pai.removeChild(filho);

// Limpar TUDO dentro de um elemento
ul.innerHTML = '';   // apaga todos os filhos de uma vez
✅ padrão da prova O professor usa ul.innerHTML = '' para limpar a lista antes de redesenhar — é a forma mais simples e é aceita na prova.

// Ou com arrow function anônima

06 Módulos ES6 (Questão 2 da prova)

Módulos ES6 permitem dividir o código em arquivos separados, cada um com sua responsabilidade. A prova pede exatamente essa organização.

Estrutura de diretórios esperada

organização de arquivos 
projeto/
├── index.html       ← HTML + <script type="module">
└── js/
    ├── app.js       ← orquestrador: importa e inicializa
    ├── dom.js       ← manipulação do DOM (criar, renderizar)
    └── storage.js   ← localStorage (salvar, buscar)
⚠️ OBRIGATÓRIO no HTML O script principal deve ter type="module", caso contrário os imports não funcionam!
<script type="module" src="js/app.js"></script>

Como exportar e importar

storage.js — exporta funções 
const KEY = 'listaTarefas';

// "export" torna a função disponível para outros arquivos
export function obterTarefas() {
  return JSON.parse(localStorage.getItem(KEY)) || [];
}

export function salvarTarefas(lista) {
  localStorage.setItem(KEY, JSON.stringify(lista));
}
dom.js — importa de storage e exporta renderizar 
// "import" traz funções de outro módulo
import { obterTarefas, salvarTarefas } from './storage.js';

export function renderizarLista() {
  const ul = document.getElementById('listaTarefas');
  ul.innerHTML = '';           // limpa antes de redesenhar
  const tarefas = obterTarefas();

  tarefas.forEach((tarefa, index) => {
    const li  = document.createElement('li');
    const btn = document.createElement('button');
    li.textContent  = tarefa.texto;
    btn.textContent = 'Excluir';
    btn.addEventListener('click', () => excluir(index));
    li.appendChild(btn);
    ul.appendChild(li);
  });
}

function excluir(index) {
  const lista = obterTarefas();
  lista.splice(index, 1);   // remove 1 item na posição index
  salvarTarefas(lista);
  renderizarLista();
}
app.js — orquestrador 
import { obterTarefas, salvarTarefas } from './storage.js';
import { renderizarLista }             from './dom.js';

const input = document.getElementById('novaTarefa');
const btn   = document.getElementById('btnAdicionar');

btn.addEventListener('click', () => {
  const texto = input.value.trim();
  if (!texto) { alert('Digite uma tarefa!'); return; }

  const lista = obterTarefas();
  lista.push({ texto, concluida: false });
  salvarTarefas(lista);

  input.value = '';
  renderizarLista();
});

// Carrega a lista ao abrir a página
renderizarLista();

import { nome } from './arquivo.js' → importa de outro módulo.
Sempre use ./ antes do nome do arquivo no import.

07 localStorage (Questão 3 da prova)

O localStorage salva dados no navegador do usuário. Os dados persistem mesmo após fechar e reabrir o navegador (ao contrário das variáveis JS, que somem ao recarregar a página).

Os 4 métodos essenciais

Salvar
localStorage.setItem(chave, valor)

Grava um par chave → valor. Valor deve ser string.

Ler
localStorage.getItem(chave)

Retorna o valor como string (ou null se não existir).

Remover 1
localStorage.removeItem(chave)

Remove apenas aquela chave do storage.

Limpar tudo
localStorage.clear()

Apaga todos os dados do domínio atual.

Armazenar objetos/arrays — JSON obrigatório

O localStorage só aceita strings. Para guardar objetos ou arrays, usamos JSON.stringify() ao salvar e JSON.parse() ao ler:

salvar e recuperar array de objetos 
// ── SALVAR ──
const tarefas = [
  { texto: 'Estudar DOM', concluida: false },
  { texto: 'Fazer prova', concluida: false }
];
// Converte array → string antes de guardar
localStorage.setItem('listaTarefas', JSON.stringify(tarefas));

// ── LER ──
// Converte string → array ao recuperar
// O "|| []" evita erro se a chave não existir ainda
const lista = JSON.parse(localStorage.getItem('listaTarefas')) || [];
❌ erro clássico Esquecer o JSON.stringify() ao salvar salva [object Object] em vez dos dados reais. Sempre converta!

Padrão de uso na prova (funções utilitárias)

funções reutilizáveis de storage 
const KEY = 'listaTarefas';

// Lê a lista do storage (retorna [] se vazio)
function obterLista() {
  return JSON.parse(localStorage.getItem(KEY)) || [];
}

// Salva a lista no storage
function salvarLista(lista) {
  localStorage.setItem(KEY, JSON.stringify(lista));
}

// ── Adicionar ──
function adicionarTarefa(texto) {
  const lista = obterLista();
  lista.push({ texto, concluida: false });
  salvarLista(lista);
}

// ── Editar ──
function editarTarefa(index, novoTexto) {
  const lista = obterLista();
  lista[index].texto = novoTexto;
  salvarLista(lista);
}

// ── Excluir ──
function excluirTarefa(index) {
  const lista = obterLista();
  lista.splice(index, 1);  // remove 1 item na posição index
  salvarLista(lista);
}
✅ dica: array.splice(index, 1) splice(index, 1) remove 1 elemento a partir da posição index. É o método padrão do professor para excluir tarefas. Memorize!

js/dom.js — responsável pela criação/remoção de elementos 
import { obterTarefas, salvarTarefas } from './storage.js';

// Renderiza TODA a lista do zero
export function renderizarLista() {
  const ul = document.getElementById('listaTarefas');
  ul.innerHTML = '';  // limpa antes de redesenhar

  obterTarefas().forEach((tarefa, index) => {
    const li  = document.createElement('li');
    const span = document.createElement('span');
    const btnEditar  = document.createElement('button');
    const btnExcluir = document.createElement('button');

    span.textContent      = tarefa.texto;
    btnEditar.textContent  = 'Editar';
    btnExcluir.textContent = 'Excluir';

    btnEditar.addEventListener('click', () => editar(index));
    btnExcluir.addEventListener('click', () => excluir(index));

    li.append(span, btnEditar, btnExcluir);
    ul.appendChild(li);
  });
}

function editar(index) {
  const lista = obterTarefas();
  const novoTexto = prompt('Novo texto:', lista[index].texto);
  if (novoTexto === null || !novoTexto.trim()) return;
  lista[index].texto = novoTexto.trim();
  salvarTarefas(lista);
  renderizarLista();  // atualiza a exibição
}

function excluir(index) {
  const lista = obterTarefas();
  lista.splice(index, 1);  // remove 1 item
  salvarTarefas(lista);
  renderizarLista();  // atualiza a exibição
}

js/dom.js

js/dom.js — criação e remoção de elementos da lista 
import { obterLeituras, salvarLeituras } from './storage.js';

export function renderizar() {
  const ul = document.getElementById('listaLeituras');
  ul.innerHTML = '';  // limpa antes de redesenhar

  obterLeituras().forEach((leitura, index) => {
    const li         = document.createElement('li');
    const btnEditar  = document.createElement('button');
    const btnRemover = document.createElement('button');

    li.innerHTML = `
      <strong>${leitura.titulo}</strong><br>
      Autor: ${leitura.autor}<br>
      Data de Início: ${leitura.data}<br>
      Status: ${leitura.status}
    `;

    btnEditar.textContent  = 'Editar';
    btnRemover.textContent = 'Remover';

    btnEditar.addEventListener('click', () => editar(index));
    btnRemover.addEventListener('click', () => remover(index));

    li.append(btnEditar, btnRemover);
    ul.appendChild(li);
  });
}

function editar(index) {
  const lista = obterLeituras();
  const l = lista[index];
  const novoTitulo = prompt('Título:',  l.titulo);
  if (novoTitulo === null) return;  // cancelou
  lista[index] = {
    titulo: novoTitulo.trim(),
    autor:  prompt('Autor:',  l.autor).trim(),
    data:   prompt('Data:',   l.data).trim(),
    status: prompt('Status (Em andamento / Concluída):', l.status).trim()
  };
  salvarLeituras(lista);
  renderizar();
}

function remover(index) {
  const lista = obterLeituras();
  lista.splice(index, 1);  // remove 1 item na posição index
  salvarLeituras(lista);
  renderizar();
}

Persistência com localStorage

Utilizando o mesmo Gerenciador de Tarefas, implemente uma versão com localStorage para persistência dos dados. A aplicação deve atender a:

  • Ao adicionar uma tarefa, ela deve ser salva no localStorage
  • Ao editar uma tarefa, as alterações devem ser refletidas no localStorage
  • Ao excluir uma tarefa, o localStorage deve ser atualizado
  • Ao recarregar a página, a lista de tarefas deve ser exibida automaticamente, recuperando os dados do localStorage

js/storage.js

js/storage.js — responsável APENAS pelo localStorage 
const KEY = 'tarefas-app';

export function obterTarefas() {
  return JSON.parse(localStorage.getItem(KEY)) || [];
  //     ↑ parse obrigatório              ↑ fallback para array vazio
}

export function salvarTarefas(lista) {
  localStorage.setItem(KEY, JSON.stringify(lista));
  //                         ↑ stringify obrigatório
}

js/dom.js

js/dom.js — criação e remoção de elementos da lista 
import { obterTarefas, salvarTarefas } from './storage.js';

export function renderizarLista() {
  const ul = document.getElementById('listaTarefas');
  ul.innerHTML = '';  // limpa antes de redesenhar

  obterTarefas().forEach((tarefa, index) => {
    const li         = document.createElement('li');
    const span       = document.createElement('span');
    const btnEditar  = document.createElement('button');
    const btnExcluir = document.createElement('button');

    span.textContent      = tarefa;
    btnEditar.textContent  = 'Editar';
    btnExcluir.textContent = 'Excluir';

    btnEditar.addEventListener('click', () => editar(index));
    btnExcluir.addEventListener('click', () => excluir(index));

    li.append(span, btnEditar, btnExcluir);
    ul.appendChild(li);
  });
}

function editar(index) {
  const lista = obterTarefas();
  const novo = prompt('Editar tarefa:', lista[index]);
  if (novo === null || !novo.trim()) return;
  lista[index] = novo.trim();
  salvarTarefas(lista);
  renderizarLista();
}

function excluir(index) {
  const lista = obterTarefas();
  lista.splice(index, 1);  // remove 1 item na posição index
  salvarTarefas(lista);
  renderizarLista();
}

js/app.js

js/app.js — orquestrador, importa e inicializa tudo 
import { obterTarefas, salvarTarefas } from './storage.js';
import { renderizarLista }             from './dom.js';

const input = document.getElementById('inputTarefa');
const btn   = document.getElementById('btnAdicionar');

btn.addEventListener('click', () => {
  const texto = input.value.trim();
  if (!texto) { alert('Digite uma tarefa!'); return; }

  const lista = obterTarefas();
  lista.push(texto);
  salvarTarefas(lista);

  input.value = '';
  renderizarLista();
});

// Ao carregar a página, exibe as tarefas salvas
renderizarLista();
